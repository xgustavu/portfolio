Sobre Mim

Olá! Sou Gustavo Carvalho, um desenvolvedor PHP Junior apaixonado por criar soluções web eficientes e elegantes. Atualmente, trabalho na Simova, onde desenvolvo sistemas e aplicações utilizando PHP.

Minha jornada na programação começou com a curiosidade de entender como a tecnologia que usamos diariamente funciona por trás das interfaces. Essa curiosidade se transformou em paixão e hoje busco constantemente aprimorar minhas habilidades e conhecimentos.

Acredito que o código limpo e bem estruturado é a base para soluções duradouras, e por isso me dedico a seguir as melhores práticas de desenvolvimento.

PHP MySQL PostgreSQL MongoDB HTML5 CSS3 JavaScript TypeScript React React Native Git

Projeto Calorias

O LifePlus é um projeto desenvolvido por nossa equipe com o objetivo de facilitar o controle de calorias. As principais funcionalidades da plataforma incluem: Cadastro de refeições,Cadastro de novas comidas, Atualização de dados, Gráficos de acompanhamento. O LifePlus visa fornecer uma solução prática e intuitiva para o acompanhamento nutricional e controle dietético.
